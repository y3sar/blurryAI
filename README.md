# BlurryAI

Blur backgrounds of images using pytorch, MaskRCNN
 
## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the necessary packages.

```bash
pip install app/requirements.txt
```

## Usage

```python
$ python app/app.py
```
